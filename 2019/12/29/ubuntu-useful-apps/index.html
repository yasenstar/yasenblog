Ubuntu Unofficial Office 365 Outlook Client – Prospect-Mail

Using default Thunder Mail, it’s fine for my personal messaging, but it’s not authorized to connect with Exchange Outlook.

Happened to know Prospect-Mail (github: https://github.com/julian-alarcon/prospect-mail), really easy to install although via snap, and usage is smoothly. As bonus, after connecting Outlook 365, you can also able to connect to other Office 365 components, e.g. Sharepoint, Planner, Team, etc.. This makes my Ubuntu client really approaching the Working Machine!

Thanks for the blog: https://websiteforstudents.com/install-prospect-mail-unofficial-outlook-365-client-on-ubuntu-18-04/

This list is tested now under Ubuntu 20.04, and will keep extended…
