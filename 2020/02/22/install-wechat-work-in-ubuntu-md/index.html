install_wechat_work_in_ubuntu.md

很遗憾，直到ubuntu20.04，其默认软件库中还是没有一些非常常用的中国的软件，比如微信，QQ，企业微信，钉钉等。

deepin作为国内在deepin中的本地化做的很充分的发行版，能够帮助到这一点。

mirrors.aliyun.com/deepin/pool/non-free/d/

在这里可以直接找各种deepin软件包下载使用dpkg -i的方式可以安装，但是会出现依赖问题需要逐个解决。

下面的方法通过统一脚本按照依赖，来在ubuntu环境下使用debian包来安装企业微信和微信，很方便，主要要解决的是依赖包的问题。

好在laitj的帖子（https://blog.csdn.net/qq_40246287/article/details/89705994?depth_1-utm_source=distribute.pc_relevant.none-task&utm_source=distribute.pc_relevant.none-task）给出了很简便的方法。

这里借鉴记录一下，多谢laitj！
