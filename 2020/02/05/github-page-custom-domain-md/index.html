Managing a custom domain for Github Page

My current github page (check from your Github repository) is github.com/yasenstar/yasenblog.

I already have one domain name I Love Coding, wish to link that in the Github Page, so that it can be accessed from my domain name.

As this site is not yet SSL enable, ensure to uncheck “force SSL”.

In the setting of Github page repository, add the www.ilovecoding.cn in the Custom Domain is the simple step.

However, it must be set one CNAME DNS record in the domain side first.

Setting like:

  • 主机记录： www
  • 记录类型： CNAME
  • 解析线路： （默认）
  • 记录值： yasenstar.github.io
